目标：本项目计划推出一个案例驱动的javascript系列教程。

实现方式：
1.对常见组件进行重写、优化、封装。
常见组件：焦点图、tab图、下拉菜单、提示框、
dir:example/

2.对常见操作的函数归类汇总。
dir:lib/
